G-value Meta-Analysis

Many measurements of the gravitational constant have been conducted over the last 200 years using different source and test mass compositions. If the gravitational constant is composition-dependent as predicted by SPD-chromogravity, then the measured values should correlate with the predicted values when sorted by material composition.

This tool lets you select which measurements to include in the meta-analysis, apply quality filters, and observe how Kendall's \(\tau_b^+\) and the p-value change in real time. The chart updates live as you toggle measurements on or off.

Note: Based on Sky Darmos (2022–2026), Strong evidence for composition-dependent gravity from Cavendish experiments. All G-values are in units of \(10^{-11}\ \text{m}^3\,\text{kg}^{-1}\,\text{s}^{-2}\).

The method

To test whether the measured G-values correlate with the predicted values, we use a modified Kendall's \(\tau_b\) — a rank-correlation coefficient that counts concordant pairs (both predicted and observed agree on ordering) minus discordant pairs, normalised by the total pairs corrected for ties. We call our variant \(\tau_b^+\) to flag one deliberate departure from the standard convention, explained below.

\[ \tau_b^+ = \frac{C - D + T_{XY}}{\sqrt{(C + D + T_X + T_{XY}) \times (C + D + T_Y + T_{XY})}} \]

Significance testing does not use \(\tau_b^+\) directly. Its exact asymptotic variance under this particular tie structure has no clean closed form, and deriving one honestly would be genuinely messy and complicated, without being any more trustworthy than the alternative below. Instead we get a conservative p-value using the simple, well-known variance of tau-a (exact when there are no ties) applied to a deliberately cautious stand-in value: the next higher tau-a.

Only the \(T = T_X + T_Y\) exclusive ties are still "ambiguous" here — joint ties (\(T_{XY}\)) are already resolved into \(C\) above, so they play no part in this step. Each exclusive tied pair could in principle have gone concordant or discordant; we search over every way the group of \(T\) ties could have shifted the numerator, and take the smallest shift whose resulting tau-a value is still strictly greater than \(\tau_b^+\):

\[ \tau_a^{\text{next}} \;=\; \min_{k} \left\{ \frac{S+k}{P} \; : \; \frac{S+k}{P} > \tau_b^+ \right\}, \quad k \in \{T, T-2, T-4, \dots\} \]

where \(S = C - D\). That value is then used with the simple tau-a standard deviation:

\[ z = \frac{\tau_a^{\text{next}}}{\sqrt{\dfrac{2(2n+5)}{9n(n-1)}}} \]

This is conservative by construction: since \(\tau_a^{\text{next}}\) is chosen as the closest tau-a value that still exceeds \(\tau_b^+\) — not the true, generally larger, worst-case tau-a — it never makes the result look more significant than it is. When there are no exclusive ties (\(T=0\)), \(\tau_a^{\text{next}} = \tau_b^+\) and this reduces to the direct formula. The page shows both values side by side ("τ-b+: … / τ-a: …") whenever they differ, so the number actually feeding \(z\) is never hidden.

\(C\) = number of concordant pairs, including joint ties (see below)

\(D\) = number of discordant pairs — pairs where the ordering disagrees

\(T_X\) = pairs tied on predicted value only  ·  \(T_Y\) = pairs tied on observed value only

\(T_{XY}\) = "joint ties" — pairs tied on both predicted and observed value

\(S = C - D\)  ·  \(P\) = total pairs \(= C+D+T_X+T_Y+T_{XY} = n(n-1)/2\)

\(n\) = number of selected measurements

The p-value is right-tailed, computed from the normal CDF of \(z\). Ties arise from measurement uncertainty, not from fundamental degeneracy.

On \(\tau_b^+\) vs. standard \(\tau_b\): standard tau-b drops a pair tied on both predicted and observed value entirely — it contributes to neither \(C\), \(D\), \(T_X\), nor \(T_Y\). Here, such a "joint tie" is instead counted as concordant, added to \(C\) and to \(T_{XY}\) — \(T_X\) and \(T_Y\) stay pure exclusive tie-counts, unaffected. Two different experiments independently landing on the same predicted and observed value is treated as supporting evidence, not as an uninformative pair. Keeping \(T_{XY}\) out of \(T_X\) and \(T_Y\) (rather than also inflating them) keeps the denominator large enough that perfect agreement can never push \(\tau_b^+\) above 1. This affects exactly two pairs in the default dataset — De Boer (1987) & Lamporesi (2008) and Goldblum (1987) & Ritter (1990).
